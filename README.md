# Proyecto Atlas

`Proyecto Atlas` es un proyecto pensado para convertirse en una plataforma pra documentación de proyectos de software, con el objetivo de facilitar la colaboración y el intercambio de conocimientos entre desarrolladores. La plataforma permitirá a los usuarios crear y compartir documentación de sus proyectos, así como acceder a la documentación de otros proyectos para aprender y colaborar.

## Características principales

- Creación de workspaces: Los usuarios podrán crear workspaces para organizar sus proyectos y su documentación de manera eficiente.
